# Event Images

Place your event images in this folder with the following names:

1. **event1.jpg** (or event1.png) - For "The Security Keynote"
2. **event2.jpg** (or event2.png) - For "Cyber Cypher-Taqneeq"
3. **event3.jpg** (or event3.png) - For "The Talk"

## Supported Formats
- JPG/JPEG
- PNG
- WebP

## Recommended Size
- Width: 800-1200px
- Height: 400-600px
- Aspect Ratio: 16:9 or 2:1 works best

## How to Add Images

1. Copy your image files to this folder (`public/events/`)
2. Rename them to `event1.jpg`, `event2.jpg`, and `event3.jpg` (or use .png extension)
3. The images will automatically appear on the events page
