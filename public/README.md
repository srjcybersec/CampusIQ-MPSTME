# Public Assets Directory

This directory contains static assets for the CampusIQ application.

## Logo File

Please place your CampusIQ logo file here with the name:
- `campusiq-logo.png` (or `.jpg`, `.webp`, `.svg`)

The logo should be:
- High resolution (at least 256x256px recommended)
- Transparent background (PNG with transparency preferred)
- Optimized for web use

The logo will be automatically used in:
- Navigation bar (top left)
- Login page
